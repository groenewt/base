# skill: games/equilibrium
**Signature** `G : (A,A') → (B,B') = (Strat_G, play_G: Strat_G → Optic,
eq_G: context → P(Strat_G))`.
**Semantics** Open games (Ghani–Hedges–Winschel–Zahn): agency as parametrised
optics; forward play, backward payoff; composition of games = composition of
optics under Para. Equilibrium is compositional — checked locally, valid
globally.
**Law** eq of a composite factors through eqs of parts (the whole game never
needs recomputation from scratch).
**Binding** Multi-agent GraphAtlas negotiation (peer sync policies, resource
scheduling) typed as open games; Bayesian variant = Bayesian lenses
(research frontier, flagged).
